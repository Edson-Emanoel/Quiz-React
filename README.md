## Comandos do Projeto

    * Comando para criar o Projeto:

    ```
        npm create vite@latest
    ```

    * Comando para instalar as dependências

    ```
        npm i
    ```
    
    * Comando para rodar o projeto

    ```
        npm run dev
    ```
